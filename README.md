# 🍳 Цифровая Книга Рецептов (Cookbook)

Современное динамическое веб-приложение для поиска, фильтрации и хранения кулинарных рецептов. Проект реализован на бэкенд-стеке **Django** с использованием чистого, адаптивного фронтенда. Разработано в качестве коммерческого кейса для личного портфолио.

## ✨ Основной функционал (Бизнес-логика)
*   **Умный поисковый движок:** Многословный поиск на базе `Q`-объектов Django. Бэкенд автоматически разбивает поисковую фразу на отдельные слова, игнорируя регистр и символы дефиса.
*   **Динамическая фильтрация:** Напольное распределение по 10 кулинарным категориям (Супы, Салаты, Завтраки, Детское меню и т.д.) через GET-запросы к базе данных.
*   **Интерактивная сортировка:** Быстрая перестройка сетки рецептов по сложности, новизне или времени приготовления.
*   **Оптимизированная пагинация:** Ограничение нагрузки на БД — вывод строго по 6 рецептов на одну страницу со сквозным сохранением активных поисковых фильтров.
*   **Автономное сессионное избранное:** Возможность добавления рецептов в персональную закладку пользователя через сессии (`request.session`) без необходимости авторизации.
*   **Рекомендательная система:** Алгоритм автоматического подбора 3 похожих блюд из той же категории на странице детального просмотра.
*   **Форма добавления рецептов (CRUD):** Полноценная валидируемая Django-форма с поддержкой загрузки медиафайлов (фотографий) для расширения кулинарной книги.
*   **Наполнение базы (Seed data):** Скрипт автоматического наполнения базы данных на 150 уникальных рецептов мировых блюд.

## 🛠 Технологический стек
*   **Backend:** Python 3.x, Django 6.x (ORM, Sessions, Paginator, Forms, MVC Architecture)
*   **Frontend:** HTML5, CSS3 (CSS-сетки, адаптивная верстка, кастомный дизайн без внешних CDN)
*   **База данных:** SQLite3

## 🚀 Как запустить проект локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   cd имя_репозитория
   ```

2. Разверните и активируйте виртуальное окружение:
   ```bash
   python -m venv .venv
   # Для Windows (PowerShell):
   .venv\Scripts\Activate.ps1
   ```

3. Установите зависимости:
   ```bash
   pip install django pillow
   ```

4. Выполните миграции и запустите сервер:
   ```bash
   python manage.py migrate
   python manage.py runserver
   ```
   Сайт будет доступен по адресу: `http://127.0.0`
